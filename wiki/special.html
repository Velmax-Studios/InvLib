Specialized Menus

Customized wrappers for non-chest inventory types.

AnvilMenu (Input)

Capture text input from players using the virtual anvil renaming slot. Includes built-in handlers for real-time rename tracking and final completion.

AnvilMenu anvil = new AnvilMenu();
anvil.onRename(text -> { /* Real-time text */ });
anvil.onComplete((text, event) -> { /* Final submit */ });

PagedMenu

Perfect for large databases or player lists. Automatically handles slot filling across multiple pages. Supports shared multi-viewer page synchronization.

PagedMenu paged = new PagedMenu(rows, contentSlots);
paged.addContent(button);
paged.createNextPageButton(); // Built-in helper

CraftingMenu

Provides a simplified 3x3 grid API for virtual crafting tables. Easily map coordinates (row, col) to crafting slots.

CraftingMenu menu = new CraftingMenu();
menu.setGridButton(1, 1, button); // Middle slot
menu.setResultButton(result);
